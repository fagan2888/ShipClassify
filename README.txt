1. 所有源代码存放在./jianmo 目录中，附件1-6数据已转化为.csv格式，放在./jianmo/data目录下，运行结果（图片、输出数据）存放在./jianmo/result目录中，其余目录为程序运行依赖文件；

2. 源代码为.py格式，在shell中键入 python *.py 即可运行，*.py文件均为文本文件，可用文本编辑器查看或修改；

3. route.py 程序可绘制信号源运动轨迹，route_revise.py 程序用于修正轨迹误差， signal_revise.py 程序用于修正信号特征数值误差， signal _classify.py用于题目三中对附件5数据进行目标识别，输出结果为目标物编号及其可能的船舶名称或船舶编号， ship_classify.py 程序用于对附件6数据进行目标识别，输出结果为目标物编号及其可能的船舶编号或船舶名称。

4.关于./jianmo/data目录下的数据说明。该目录下存储了一些列*.csv文件，这些文件源自附件中的数据，进行此转化的目的是方便程序处理，Excel表格附件m中的第n个数据表转化并命名为mn.csv,如61.csv文件对应的是附件6中第2个数据表， *.csv是文本文件，可直接用文本编辑器查看。
